# Steps

## Overview

## retrieve

Retrieves data for a specific step in a workflow

### Example Usage

```python
from novu_py import Novu


with Novu(
    secret_key="YOUR_SECRET_KEY_HERE",
) as novu:

    res = novu.workflows.steps.retrieve(workflow_id="<id>", step_id="<id>")

    # Handle response
    print(res)

```
