# MarkAs

Mark all subscriber messages as read, unread, seen or unseen

## Example Usage

```python
from novu_py.models import MarkAs

value = MarkAs.READ
```


## Values

| Name     | Value    |
| -------- | -------- |
| `READ`   | read     |
| `SEEN`   | seen     |
| `UNREAD` | unread   |
| `UNSEEN` | unseen   |
