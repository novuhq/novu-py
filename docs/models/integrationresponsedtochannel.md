# IntegrationResponseDtoChannel

The channel type for the integration, which defines how it communicates (e.g., email, SMS).

## Example Usage

```python
from novu_py.models import IntegrationResponseDtoChannel

value = IntegrationResponseDtoChannel.IN_APP
```


## Values

| Name     | Value    |
| -------- | -------- |
| `IN_APP` | in_app   |
| `EMAIL`  | email    |
| `SMS`    | sms      |
| `CHAT`   | chat     |
| `PUSH`   | push     |
